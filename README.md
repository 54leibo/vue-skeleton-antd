# vue-skeleton-antd
vue 构建的 ant design 风格骨架屏组件

## API

### Skeleton 骨架屏

|  属性 |  说明 | 类型  |  默认值 |
| ------------ | ------------ | ------------ | ------------ |
|  active |  是否展示动画效果 | boolean  | true  |
|  avatar |  是否显示头像占位图 |  boolean |  true |
|  paragraph |  是否显示段落占位图 |  boolean |  true |
|  title |  是否显示标题占位图 |  boolean |  true |
|  loading |  为 `true` 时，显示占位图。反之则直接展示子组件 |  boolean |  - |
