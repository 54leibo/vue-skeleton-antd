# 快速上手

# 安装

使用 npm 安装

npm i vue-skeleton-antd

或使用 yarn 安装

yarn add vue-skeleton-antd

# 导入组件

在 main.js 中导入之后，就可以当做全局组件使用：

import Vue from 'vue';
import Skeleton from 'vue-skeleton-antd';

Vue.use(Skeleton);
